# student-management-system-python

Student Management System

I am a first-year Engineering student from the AI & Data Science (AI & DS) branch.
This project is a Student Management System built using Python, where users can manage student records efficiently.

The system allows users to store and manage:
Personal details
Contact information
Academic details

This is my first project in my coding journey during the first semester of college.

Features :
Add new student records
Update existing student information
Delete student records
Search student details

Technologies Used:
Python
JSON (for data storage)
File Handling

This project is created for learning purposes and to understand the basic concepts of the Python programming,
